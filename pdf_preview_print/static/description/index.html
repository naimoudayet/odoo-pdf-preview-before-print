PDF Preview Before Print

Preview any PDF report before printing or downloading

Odoo 16 Ready Zero Config Keyboard Shortcuts

The Problem

Every time you click Print in Odoo, the PDF downloads directly to your device. Wrong template? Blank page? Missing data? You only find out after wasting paper or hunting through your downloads folder.

Key Features

Full-Screen Preview

A clean OWL dialog shows the full PDF before any action is taken. Scroll through multi-page documents effortlessly.

Print from Preview

One click or press P to send the document directly to your printer from the preview.

Download from Preview

Press D or click Download to save the PDF after confirming it looks correct.

Before vs After

Without This Module

  • Click Print → PDF downloads immediately
  • No chance to review before printing
  • Wrong template? Wasted paper
  • Blank page? Hunt through downloads
  • Batch print errors go unnoticed

With PDF Preview

  • Click Print → Full preview opens first
  • Review the document before acting
  • Print, Download, or Cancel from dialog
  • Keyboard shortcuts: P / D / Esc
  • Works with single and batch reports

Installation

1

Download & Extract

Download the module and extract it to your Odoo addons directory.

2

Update Apps List

Go to Apps menu and click "Update Apps List".

3

Install

Search for "PDF Preview Before Print" and click Install.

Done!

All PDF reports now show a preview before printing. No setup needed.
