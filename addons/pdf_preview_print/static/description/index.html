PDF Preview Before Print

Preview PDF reports before printing or downloading — backend & portal

The Problem

Every time you click Print or Download in Odoo, the PDF is silently pushed to your printer or downloads folder. There is no way to glance at it first. Wrong template? Blank page? Missing data? You only find out after wasting paper or hunting through your downloads. Portal customers face the same issue — they must download an invoice just to see if it is the right one.

Features

Full-Screen Preview

A clean, full-screen OWL dialog shows the PDF before any action is taken. Scroll through multi-page documents effortlessly.

Print from Preview

One click (or press P) to send the previewed document directly to the printer — no extra steps.

Download from Preview

Press D or click Download to save the PDF after you have confirmed it looks correct.

Keyboard Shortcuts

P → Print • D → Download • Esc → Close. No mouse required.

Portal Support

External customers see a Preview button next to every invoice, quotation, and purchase order in their portal.

Zero Configuration

Install the module and it works immediately. No settings page, no configuration wizard, no per-user setup.

How It Works

  1. Backend: The module patches the default report download action. Instead of downloading the PDF immediately, it opens an OWL dialog with an embedded PDF viewer. Users then choose to Print, Download, or Close.
  2. Portal: A lightweight JavaScript module adds a Preview button next to every PDF download link on portal pages. Clicking it opens the same full-screen viewer.
